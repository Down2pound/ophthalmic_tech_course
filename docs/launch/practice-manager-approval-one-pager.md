# OptiTech Academy Practice Manager Approval One-Pager

Use this when a practice manager, supervisor, or training lead wants to explain
OptiTech Academy to an owner, administrator, or budget decision-maker.

Simple translation: this is the short "why this is worth buying" page.

## Short Summary

OptiTech Academy is a self-paced ophthalmic technician foundations course for
new ophthalmic technicians, medical assistants moving into eye care, and
practices that want a more consistent onboarding starting point.

This is not a certification program, and it does not replace local supervision,
clinical policy, hands-on observation, or competency signoff. It gives learners
plain-language foundations before the practice teaches its own workflow.

## Why This May Help The Practice

- New hires can learn common ophthalmic vocabulary before clinic shadowing.
- Supervisors can use shared language when giving feedback.
- Medical assistants can build eye-care confidence before taking on specialty
  tasks.
- The Skills Passport can help organize supervised practice conversations.
- Managers can give every new learner the same starting point instead of
  repeating the same basic explanations from scratch.
